## 📁 How to Run Locally

> Make sure you have **Node.js** and **npm** installed.

```bash
# Step 1: Clone the repository
git clone https://github.com/Epochalypse00/Aria_Portfolio.git
cd Aria_Portfolio

# Step 2: Install dependencies
npm install

# Step 3: Run the development server
npm run dev
